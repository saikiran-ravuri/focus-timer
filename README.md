# 📖 About

Focus Timer is a lightweight productivity web application inspired by the Pomodoro Technique. It provides a clean and intuitive interface for managing focus sessions using a countdown timer, animated circular progress indicator, and adjustable session duration.

This project was built to strengthen my understanding of **JavaScript fundamentals, DOM manipulation, state management, timers, responsive design, and Git/GitHub workflow**.

---

# ✨ Features

- ⏱️ Countdown Timer
- ▶️ Start, Pause & Reset Controls
- ⭕ Animated Circular Progress Ring
- 📝 Adjustable Timer Duration (1–120 Minutes)
- 📊 Live Session Status Updates
- 🔊 Completion Sound Notification
- 📱 Fully Responsive Design
